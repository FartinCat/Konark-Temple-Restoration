# Konark Sun Temple — 3D Reconstruction

A full 3D reconstruction of the Konark Sun Temple (Kalinga "Pidha Deula" style), built entirely in Blender.

<video src="https://raw.githubusercontent.com/FartinCat/Konark-Temple-Restoration/main/Output/konark_temple_cine_flythrough.mp4" controls width="100%"></video>

Also posted on X: https://x.com/FartinCat/status/2075122267395076245?s=20

## About this project

This entire project was generated by **Claude Fable 5**, in high-thinking mode, using **Claude Cowork**. The model:

- Recreated the temple's entire structural design from a **single palm-leaf manuscript** ([`Reference_Images/Konark_palm_leaf_manuscript.jpg`](Reference_Images/Konark_palm_leaf_manuscript.jpg)) sourced from Wikipedia.
- Cross-referenced additional historical photographs, sketches, and drawings of the temple, also sourced from Wikipedia.
- Used structural and architectural facts from the [UNESCO World Heritage listing](https://whc.unesco.org/en/list/246) (e.g. 24 wheels, jagamohana wall height, three-tier pidha roof) to derive accurate proportions.
- Procedurally generated the geometry in Blender via Python scripting, including golden-ratio-driven tier heights and wheel placement.
- Rendered the final cinematic flythrough video frame by frame.

No manual modeling was performed — every mesh, proportion, and camera move in this repository was authored by the model from the reference material alone.

## File structure

```
.
├── Blends/                  # Blender project files
│   └── konark.blend
├── Output/                  # Final rendered deliverables
│   └── konark_temple_cine_flythrough.mp4
├── python-scripts/          # Blender Python automation
│   ├── konark_blockout.py   # Procedural structural blockout (Jagamohana + wheels)
│   └── render_loop.py       # Batch frame-render automation
├── Reference_Images/        # Historical reference material (Wikipedia, UNESCO)
│   ├── Konark_palm_leaf_manuscript.jpg   # Primary manuscript used for reconstruction
│   └── ...                  # Historical photographs, sketches, and drawings
├── LICENSE
└── README.md
```

## Video naming convention

Deliverables in `Output/` follow `konark_<subject>_<type>.mp4`, e.g. `konark_temple_cine_flythrough.mp4` for the cinematic flythrough render.
